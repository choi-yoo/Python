# Set
```py
s = {1, 3.14, 'a'}

s[1] # TypeError: 'set' object is not subscriptable
```
- 공집합 생성시에도 반드시 set()으로 생성해야 함
    - {}로 생성할 경우, 빈 사전이 되기 때문 
- 특징
    1. 원소의 중복 허용X
    1. 원소의 순서 존재X
        - 원소의 index가 없음

## 집합의 연산
- 교집합
    ```py
    s1 = {1, 2, 3, 4, 5}
    s2 = {3, 4, 5, 6, 7}

    s1 & s2 # 출력 : {3, 4, 5}
    ```

- 합집합
    ```py
    s1 = {1, 2, 3, 4, 5}
    s2 = {3, 4, 5, 6, 7}

    s1 | s2 # 출력 : {1, 2, 3, 4, 5, 6, 7}
    ```

- 차집합
    ```py
    s1 = {1, 2, 3, 4, 5}
    s2 = {3, 4, 5, 6, 7}

    s1 - s2 # 출력 : {1, 2}
    s2 - s1 # 출력 : {6, 7}
    ```

- uniqueness 활용
    ```py
    L1 = [1, 2, 2, 3, 1, 4, 5, 6, 1, 2, 4, 6]
    L2 = [22, 33, 11, 21, 1, 22, 33, 12, 33, 24]

    # 두 숫자더미에 겹치는 종류 파악 --> 교집합!
    set(L1) & set(L2) # 출력 : {1}
    ```
