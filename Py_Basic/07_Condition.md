# Condition Statement

## If Statement
- `if`와 `:` 사이에 `조건식`이 들어감
- 들여쓰기한 부분만 if문 안의 코드
```py
money = 500
if money == 300:
    print("Coffee")
elif a > 300:
    print("Coffee")
    print("거스름돈: %d원" % (money - 300))
else:
    print("거스름돈: %d원" % (money))
```
